# 📄 DocSpace

**DocSpace** is a collaborative document editing platform inspired by Google Docs. Create, edit, and share documents with others — all in real-time.

Built with:
- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Query](https://tanstack.com/query/latest)
- [Vitest](https://vitest.dev/) for testing

---

## ✨ Features

- 📝 Create and manage rich-text documents
- 🔗 Share documents with other users (view/edit permissions)
- 👥 Connect with people (collaboration and contact system)
- 🔍 Full-text search for documents
- ⚡ Fast and responsive UI
- 💾 Auto-saving and versioning (WIP)
- 🔐 Auth with Kinde (or your chosen provider)
